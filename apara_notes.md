# Apara.design Design Analysis

## Overall Aesthetic
- White/light background (we keep dark — #0a0a0a)
- Extremely generous whitespace — sections breathe
- No visual clutter — every element earns its place

## Typography
- Massive, bold serif-style headings — very large, tight tracking
- Short, punchy headline phrases ("Not loud. Just undeniable.")
- Small eyebrow labels above headings (e.g. "Our process", "Before you decide")
- Body text is small, centered, muted — never competing with headlines
- Mix of very large display text + small supporting copy

## Layout Patterns
- Full-width sections with generous vertical padding (100px+)
- Split cards: dark left panel (text) + light/image right panel
- Floating product screenshots/images scattered around hero (not boxed)
- 3-column feature cards with image top, label tag, description below
- Process steps as simple numbered list with thin dividers
- Pricing cards: clean, minimal, price prominent, feature list with checkmarks

## Key UI Details
- Pill-shaped CTA buttons with arrow icons (→ or ⊙)
- Small category labels as pill tags (e.g. "Website & Platform")
- Thin horizontal rule dividers between sections
- Cards have very subtle borders, almost no shadow
- "Likes" counter on project cards (social proof micro-detail)
- Sticky minimal nav — logo left, links right, no background until scroll
- Footer: simple 2-column, minimal links

## What to Adapt for TrainedBy (dark theme)
- Keep dark background, use same generous whitespace
- Hero: massive bold headline + small eyebrow ("For Personal Trainers in UAE")
- Floating trainer profile card mockups around hero (like Apara's floating app icons)
- Split feature cards: dark panel text + trainer photo right
- Process section: numbered steps with thin dividers
- Pricing: clean minimal cards, AED price prominent
- Social proof: trainer count, session count as animated numbers
- Pill CTAs: "Get Your Page →" and "See a Live Profile ⊙"
- Small section labels above every heading
